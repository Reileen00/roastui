## One Stack Simple SaaS Boilerplate
The boilerplate is explained in details on this [YouTube video](https://youtu.be/tr73GylvGA0?si=fKNrIMc6fPlRbMMT).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
